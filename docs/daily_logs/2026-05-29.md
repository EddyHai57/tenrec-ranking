## 已完成事项

- 新增 `Din` 模型，语义限定为 static user-level hist snapshot + target-dependent attention。
- DIN 使用同一份 `item_id` embedding 处理 target item 和 `hist_item`。
- Attention 输入为 `[target_emb, hist_emb, target_emb * hist_emb, target_emb - hist_emb]`，输出每个 hist 位置的标量权重。
- Attention 不做 softmax；仅对 padding index `1` mask 为 0，OOV index `0` 不 mask。
- 训练层接入 metadata 中的 `sequence_features`，使 CSV / tensor loader 的 `hist_item` 能传入 DIN。
- 新增 `configs/din_smoke.yaml`，指向 `outputs/preprocessed/ctr-610578df3be5/metadata.json`。
- 新增 `tests/test_din.py`，覆盖共享 embedding、padding/OOV mask、target-dependent attention、all-padding history、attention 非全等和 forward shape。
